UAP Summoning Signal

This website generates a blend of continuous and periodic tones—including sub-audible modulations and high-frequency “dog whistle” sounds—to mimic a set of rumored “UAP summoning” signals. Each tone is carefully constructed using the Web Audio API and can be toggled on or off with the button above.

528 Hz (Harmonic) Considered a “healing” or “DNA repair” frequency in some theories. Provides a strong mid-range tone. Matches tweet requirement for a 528 Hz “harmonic” presence.
17 kHz (Ultrasonic Ping) Adds a near-ultrasonic tone above human hearing range, allegedly heard by animals or used in “dog whistles”. Matches the “ultrasonic” or high-frequency element specified in the tweet.
2.5 kHz (Chirps every 10 s) Periodic short bursts of mid-high frequency, creating a “ping” or “chirp” to add a distinct signal flare. Matches the tweet’s mention of intermittent chirps around 2.5 kHz.
432 Hz (Ambient Pad) Often regarded as a more “natural” or “calming” reference frequency, providing a warm bed of sound. Aligns with tweet reference to the 432 Hz ambient tone.
White Noise + LFO “Breath” Layer White noise softly modulated at a low rate to simulate a subtle “breathing” or “windy” backdrop. Adds a natural-sounding layer to the signal, as described in the tweet’s “breath” effect requirement.
